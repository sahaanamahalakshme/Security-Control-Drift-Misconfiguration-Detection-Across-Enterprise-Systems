# SentinelDNA — Security Control Drift & Misconfiguration Detection

> **AI-powered, real-time detection of security control drift and misconfiguration across enterprise systems** — with explainable remediation, compliance impact, and executive reporting.

Built as a hackathon project demonstrating a full 16-stage AI security pipeline, from raw event ingestion to Slack-dispatched remediation actions.

## Quick Start

### 1 — Clone & set up environment

```powershell
git clone <repo-url>
cd secure_baseline_updated

# Copy env template and add your keys
copy .env.example .env
# Then edit .env with your real GEMINI_API_KEY and HF_API_TOKEN
```

### 2 — Backend (FastAPI)

```powershell
cd backend
pip install -r requirements.txt

# Start the API server
uvicorn api.main:app --reload --port 8000

# Generate initial drift data (run once)
curl -X POST http://localhost:8000/pipeline/run
```

### 3 — Frontend (React + Vite)

```powershell
cd frontend/web
npm install
# .env already has VITE_API_BASE_URL=http://localhost:8000
npm run dev
# Opens at http://localhost:5173
```

> **Offline mode**: The frontend works without the backend — it falls back to realistic demo data in `mockData.js` and shows a "Demo data" badge.

## API Keys

All keys go in **`secure_baseline_updated/.env`** (never commit this file):

| Variable | Where to get it | Used in |
|---|---|---|
| `GEMINI_API_KEY` | [aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey) | `backend/config/llm_config.py` — primary LLM for incident storytelling |
| `HF_API_TOKEN` | [huggingface.co/settings/tokens](https://huggingface.co/settings/tokens) | `backend/config/llm_config.py` — fallback LLM (Mistral-7B) |
| `SLACK_WEBHOOK_URL` | [api.slack.com/apps](https://api.slack.com/apps) → Incoming Webhooks | `backend/remediation/webhook_dispatcher.py` — leave blank for mock/console mode |

## The 16-Stage Pipeline

| Stage | Name | Module |
|---|---|---|
| 0 | Event Ingestion | `datasets/events.json` |
| 1 | Drift Detection | `detection_engine/pipeline.py` |
| 2 | Adaptive Trust + TTL | `trust/`, `ttl/` |
| 3 | Deep Forensics (ML) | `intelligence_engine/deep_forensics.py` |
| 4 | Attack Graph | `attack_graph/graph_builder.py` |
| 5 | AI Security Tribunal | `tribunal/ai_tribunal.py` |
| 6 | Counterfactual Simulator | `intelligence_engine/counterfactual_engine.py` |
| 7 | Drift DNA | `intelligence_engine/drift_dna.py` |
| 8 | Immune Memory | `intelligence_engine/immune_memory.py` |
| 9 | Predictive Forecasting | `intelligence_engine/forecasting.py` |
| 10 | Business Impact (₹) | `intelligence_engine/business_impact.py` |
| 11 | Security Credit Score | `intelligence_engine/credit_score.py` |
| 12 | Security Time Machine | `intelligence_engine/time_machine.py` |
| 13 | AI Incident Storytelling | `intelligence_engine/storytelling.py` |
| 14 | Auto-Healer | `remediation/auto_healer.py` |
| 15 | Slack Webhook Dispatch | `remediation/webhook_dispatcher.py` |

Run all stages at once:
```powershell
cd backend
python -m intelligence_engine.run_all
```
